Project

기존에 있는 사이트를 새롭게 리디자인하고 스크립트를 사용하여 구현

직관적이고 쉬운 탐색을 위해 메뉴 구조를 재정립하고, 중요한 정보와 기능에 더 빠르게 접근할 수 있도록 배치하였으며, 전체적으로 세련된 디자인 감각을 향상시켰습니다.

  • 타입: web_redesign
  • 기여도: 개인 프로젝트 100%
  • 작업기간: 23.05.08 ~ 05.16

Style Guide

Concept

  • 세련된 디자인의 깔끔함

Keyword

  • #맑음
  • #세련된
  • #깨끗함

Typography

  • Noto sans kR - 한글 기본 서체
  • Montserrat - 영어 기본서체

Publishing

  • swiper을 이용한 슬라이드 구현
  • flex를 이용한 웹페이지 레이아웃 제작/제어
  • CSS를 사용한 hover 효과
  • position 속성을 사용한 위치 조정

Problems & Solution

  • Flex를 어떻게 사용해야 하는가!

  • 사이트를 처음 만들다 보니 flex사용이 헷갈렸다.

    div를 사용해 구간나눈후 묶어서 html 코드를 짰다. 구간을 잘 나눠 놓으니 flex 사용의 어려움이 사라졌다. flex의 사용이 많아 class값으로 지정해두고 사용하니 더욱 편리해졌다.

  • 폰트와 폰트 사이즈를 어떻게 해야할까?

  • 폰트 사용에 있어 고민을 정말 많이했다.

    폰트와 사이즈 행간에 따라 사이트의 이미지가 변화하기 때문에 고민을 많이 하였다. 최대한 깔끔하면서 브랜드 이미지와 맞는 폰트를 찾아보았다.

    ‘Montserrat’ 와 ‘Noto Sans KR’을 사용하였고, 원했던 느낌의 사이트가 구현되었다.

Review

웹 리디자인을 처음 하다보니 디자인을 어떻게 해야되는지조차 감이 오지 않았습니다. 그래서 기존에 있는 다른 사이트들을 보고 비슷한 구조로 디자인을 해 보았고, 여백과 텍스트의 위치를 고민해 보았습니다.

깔끔하지만, 정보전달에 있어서는 확실하게 디자인을 하기 위해 폰트와 폰트 사이즈 사용에 신중했습니다. flex를 많이 사용하는 구조이다 보니 크기값을 줄때 처음에 잘 되지않았지만, 한번 해보니 사용하는데 있어 어려움이 사라졌습니다.

웹디자인을 할때 폰트와 폰트 사이즈 이미지는 어떤 위치에 어떻게 사용하는지에 대해 고민을 정말 많이 하였고, 완성을 한후 제가 원했던 깔끔하고 세련된 디자인이 구현된거 같아 만족스럽습니다.
